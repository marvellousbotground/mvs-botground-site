🔒 Privacy Policy

Marvellous Verification Privacy Policy

Marvellous Verification stores only the information necessary to link Roblox accounts with Discord profiles and provide verification-related features.

  • Discord User ID
  • Roblox User ID
  • Roblox Username
  • Verification Status

How We Use This Information

Stored information is used exclusively for account linking, verification systems, and features provided by Marvellous Bot.

Data Security

We do not store passwords, authentication credentials, access tokens, or any other sensitive account information.

Data Removal

Users may remove their linked Roblox account and permanently delete their stored verification data at any time through the verification system. Additional deletion requests may be submitted by contacting the Marvellous Bot administrators.
